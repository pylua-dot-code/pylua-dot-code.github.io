Resetting OnBoot in Gorilla Updater

Step 1:

Delete the file called OnBoot.bool, located in the Variables Folder

Step 2:

Recreate the OnBoot.bool file and leave it blank, if you selected no, you are done

Step 3: (if you selected yes)

Go to this folder on your computer: C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp.
Find the file called Updater.py and delete it. Now you are done, and you can run the program again.
